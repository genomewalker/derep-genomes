A simple genome de-replication tool with fastANI

## Installation

Clone the repo:

```bash
git clone https://github.com/genomewalker/derep-genomes.git
cd derep-genomes
```


Create a conda env:

```bash
conda env create -f environment.yaml
conda activate genome-derep
```

Install it:
```bash
python setup.py install 
```
